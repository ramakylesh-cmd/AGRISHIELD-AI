Train a plant disease model — works without internet
1
Go to Teachable Machine
Open this free tool in a new tab:
👉 teachablemachine.withgoogle.com/train/image
2
Create an Image Project
Add classes like Healthy, Blight, Rust, Mildew. Upload 20–50 photos per class. Click Train Model.
3
Export → Upload to cloud
Click Export Model → TensorFlow.js → Upload (shareable link). You'll get a URL like:
https://teachablemachine.withgoogle.com/models/XXXXXXXX/
4
Paste your model URL here
Include the trailing slash. Then click Save.
💡 Tip: Once loaded the model is cached in your browser. You can go offline and it still works.
